# [Porpoise; a motorsport calendar](https://porpoise.benjami.in/)

> [!NOTE]
> This project is still in the early stage of development. It is not ready for
> production use yet.

Porpoise is a website that shows motorsport events from around the world. It is also forked from [Hollo] which enables federation through [ActivityPub], which means that you can follow Porpoise feeds from other software that supports ActivityPub like
Mastodon, Misskey, and so on.

[Hollo]: https://github.com/dahlia/hollo
[ActivityPub]: https://www.w3.org/TR/activitypub/
